# 1. Introducción
Se trata de un pequeño proyecto de Android en Kotlin. Al inicial la App se llama un servicio que devuelve información relativa a personajes de Marvel, la cual es pintada mediante un RecyclerView al iniciar.

Si se pulsa en cualquier personaje, se cargará una nueva pantalla con más información del mismo.

# 2. Tecnologías
- MVVM
- Kotlin
- Servicios REST - Retrofit
- Inyección de dependencias - Koin
- Otras librerias: Gson, Picasso, androidx

# 3. Probar
Se recomienda probar la aplicación utilizando el buildVariant "devDebug", ya que es en ese donde se han realizado las pruebas.
